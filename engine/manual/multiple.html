Using POPFile with Multiple Addresses

Provided you don't use Secure Password Authentication (and you probably don't), POPFile will automatically cope with multiple email accounts; you don't need to bother. However, POPFile can't maintain more than one set of buckets for multiple accounts - all emails get filed into the same buckets.
