設立郵筒

POPFile 是個究極萬用的程式. 妳可以叫她祇要分出垃圾信跟非垃圾信就好, 也可以讓她幫妳區分工作, 家庭, 與垃圾郵件, 或者自動地在大量不同的專案郵件間加以區分.

所謂的郵筒正如其名 - 就是某個妳的電子郵件所能被分類進去的資料匣. 妳可以拿一個郵筒來裝垃圾信, 另一個來裝一般的郵件, 或者一個拿來裝個人郵件, 一個裝工作郵件, 一個拿來裝垃圾信. 完全看妳高興.

如果妳纔剛裝好 POPFile 然後正第一次看到她的網頁介面的話, 妳將會看到一個空空如也的 '歷史' 頁面. 請在介面最上方的 '郵筒' 標籤上按一下.

在設立郵筒前, 讓咱們先假設妳想要有三個郵筒: 工作 (work), 個人 (personal) 和垃圾 (spam). 在 '郵筒' 頁面下方不遠處, 有個 '維護' 標籤, 一旁有個 '用這個名稱建立郵筒:'. 依序建立郵筒 - 記住, 妳可以隨意加以訂做 - '垃圾 (spam)' 和 '真實 (real)', 或 '工作 (work)', '家庭 (home)', 跟 '垃圾 (spam)' - 或者妳的某個專案名稱.

(注意: 有越多的郵筒的話, 就得花越多時間來調教 POPFile - 但這也將可能使她更有用.)

當妳建立了所有想用的郵筒後, POPFile 就已經準備好要開始分類郵件了. 現在是讓妳的電子郵件程式跟 POPFile 交談的時候了.
