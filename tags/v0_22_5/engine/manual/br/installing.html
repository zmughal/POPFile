Instalando o POPFile

O POPFile vem em duas versões - uma versão para Windows fácil de instalar, e uma versão multi-plataforma para usuários técnicos. Uma vez instaladas, as duas versões são idênticas.

Versão para Windows Fácil de Instalar

Faça o download do instalador para Windows na Página de Download do POPFile, exucute setup.exe e siga as instruções. Você pode executar o POPFile pelo Menu Iniciar (Iniciar > Programas > POPFile > Run POPFile in Background), ou reinicializar o computador e ele vai iniciar automaticamente. Nota importante: não vai aparecer nada na tela! Para conferir se ele está rodando corretamente, leia a seção "Accessando o POPFile" adiante.

Requisitos do Sistema

Você vai precisar de algumas coisas para começar a usar o POPFile.

  1. O último release do POPFile.
  2. Uma conta de email que usa o protocolo POP3 (a maioria das contas usam, embora você não possa usar o POPFile com serviços baseados em web-mail como Hotmail and Yahoo! Mail sem um software extra)
  3. Em torno de 2MB de espaço livre em disco

Você pode obter o último release do POPFile como um arquivo zip no topo da lista de pacotes na Página de download do POPFile. Usuários Windows preferirão o arquivo da seção chamada Windows Only.

O POPFile em si ocupa apenas pouco mais de um megabyte. A lista de palavras (chamada de corpus) usada para classificar seus emails vai ocupar algum espaço adicional dependendo de quantos emails você usar para treinar o POPFile e de quantos buckets você criar.

Versão Multi-Plataforma

Deixe o Perl rodando em sua máquina, então faça o download do POPFile Perl zip da Home Page do POPFile, e extraia-o para um diretório a sua escolha. Nas plataformas Windows, execute-o com o comando perl popfile.pl.

Verificando se o POPFile está rodando corretamente

O POPFile não tem uma interface tradicional como a maioria dos programas - você o acessa através de seu navegador web. Não se preocupe - a menos que você diga o contrário, ninguém de fora de sua máquina poderá acessá-lo.

Para carregar o POPFile, vá em http://127.0.0.1:8080, ou (para a versão Windows) clique no link 'POPFile Web Interface' no Menu Iniciar. Nota Importante: se durante o processo de intalação no Windows você escolheu uma porta diferente para comunicar-se com o POPFile, substitua a parte "8080" com a porta que você escolheu.

Se aparecer uma mensagem de erro, verifique se o POPFile está realmente rodando, e que você tem o número da porta correto (deveria ser 8080, mas se você o alterou, você precisará trocá-lo no endereço web também.)
