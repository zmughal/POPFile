安裝 POPFile

POPFile 有兩種版本 - 一種是容易安裝的 Windows 版, 另一種是給技術人員安裝的跨平台版. 不過裝好之後, 這兩個版本將會一模一樣.

容易安裝的 Windows 版

從 POPFile 下載頁面取得 Windows 安裝檔, 然後執行 setup.exe 並按照那裡的指示繼續下去. 妳可以從開始選單 (開始 > 程式 > POPFile > Run POPFile in Background) 來啟動 POPFile, 或者重新開機後她就會自動跑起來. 重要: 啥也不會出現在妳的螢幕上! 如果妳想要檢查看看她是否正確地被執行了, 請閱讀底下的 "存取 POPFile" 段落.

系統需求

妳需要一些東西纔能讓 POPFile 開始動起來.

  1. 最新釋出的 POPFile.
  2. 使用 POP3 通訊協定的電子郵件帳號 (多數的帳號都是; 不過如果沒有額外的軟體相助, 妳將不能搭配 POPFile 一併使用像是 Hotmail 和 Yahoo! Mail 這類的網頁郵件服務)
  3. 大約 2MB 左右的硬碟空間

妳可以在 POPFile 下載頁面套件清單的最上面取得最新釋出的 POPFile 的 zip 格式壓縮檔. Windows 使用者祇會想要抓被放在 Windows 檔區裏的檔案.

POPFile 本身祇會佔據大約 1 MB 再多一點空間而已. 用來分類電子郵件的字表 (又被稱做屍體) 還會佔掉額外的一些空間, 不過這會依妳如何調教 POPFile, 以及妳所建立的郵筒數量而有所不同.

跨平台版

首先要在妳的機器上把 Perl 給跑起來, 然後從 POPFile 首頁下載 POPFile Perl 腳本的 zip 壓縮檔, 並解開到妳所指定的目錄裏. 在 Windows 平台上的話, 就用 perl popfile.pl 這個指令來加以啟動.

檢查 POPFile 是否正確地跑起來了

POPFile 沒有多數程式的那種傳統介面 - 妳其實是用妳自己的網頁瀏覽器來對她加以存取的. 別擔心 - 除非妳首肯, 否則妳機器外的任何人都沒辦法碰到她.

要載入 POPFile 的話, 請連到 http://127.0.0.1:8080, 或者 (對 Windows 版的使用者來說) 在開始選單裏的 'POPFile Web Interface' 按一下. 重要: 如果, 在 Windows 的安裝過程中, 妳選擇要用其他的連接埠來跟 POPFile 交談的話, 請把這裏的 "8080" 換成任何妳所挑選的數字.

如果妳得到錯誤訊息的話, 請先檢查 POPFile 是否真的已經跑起來了, 然後看看連接埠號碼是否正確 (她應該要是 8080 纔對, 但是如果妳曾變更過, 那麼妳就得一併變更妳所要連的網址.)

如果妳看到 POPFile 畫面的話, 恭喜啦! 現在妳可以開始設定要如何分類郵件了. 讓咱們繼續設定郵筒吧. >>
