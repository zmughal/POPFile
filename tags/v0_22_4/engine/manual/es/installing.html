Instalando POPFile

POPFile viene en dos versiones - una versión para Windows facil de instalar, y otra versión multi-plataforma para usuarios con inquietudes tecnicas. Una vez instalado, las dos son identicas.

Versión para Windows 'facil de instalar'

Descargue el instalador para Windows desde la Página de descarga de POPFile, ejecute setup.exe y siga las instrucciones que se le presenten. Puede iniciar POPFile desde el Menu de Inicio (Inicio > Programas > POPFile > Run POPFile in Background), o reiniciar y arrancará automaticamente. Nota Importante: no se presentará nada en su pantalla! Para comprobar que está funcionando correctamente, lea mas abajo la sección "Accediendo a POPFile".

Necesidades del sistema

Necesitará varias cosas para empezar con POPFile.

  1. La última versión de POPFile.
  2. Una cuenta de correo que utilize el protocolo POP3 (la mayoria de las cuentas lo usan, pero no se puede usar POPFile con servicios basados en web como Hotmail y Yahoo! Mail sin utilizar programas extra)
  3. Unos 2MB de espacio libre en el disco

Puede obtener un archivo zip con la última versión de POPFile al principio de la lista de paquetes de la Página de descarga de POPFile. Los usuarios de Windows necesitan solo el archivo de la sección Windows.

POPFile ocupa sobre un mega. La lista de palabras (llamada el corpus) que se utiliza para clasificar su correo necesitará espacio adicional dependiendo de cuanto correo use usted para entrenar POPFile y cuantas categorías cree.

Versión Multi-Plataforma

Consiga que Perl funcione en su ordenador, descargue despues el archivo zip de POPFile Perl desde la Página de POPFile, y extráigalo a una carpeta. En las plataformas Windows, arránquelo mediante el comando perl popfile.pl.

Comprobando que POPFile está funcionando correctamente

POPFile no tiene un interface tradicional como la mayoría de los programas - podrá acceder a él mediante su navegador web. No se preocupe - a menos que usted lo decida, nadie podrá acceder a él desde fuera de su ordenador .

Para cargar POPFile, vaya a http://127.0.0.1:8080, o (para la versión de Windows) clic en el acceso directo 'POPFile Web Interface' de su Menu de Inicio. Nota Importante: si, durante el proceso de instalación en Windows, usted elije un puerto diferente para comunicarse con POPFile, cambie la parte "8080" con lo que haya elegido.

Si recibe un mensaje de error, compruebe que POPFile está funcionando de verdad, y que usted ha puesto bien el número de puerto (debeberíaser 8080, pero si usted lo ha cambiado, necesitará cambiarlo también en la dirección web a la que quiere acceder.)
