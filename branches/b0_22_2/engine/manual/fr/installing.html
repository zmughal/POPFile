Installation de POPFile

POPFile existe en deux versions : une version Windows facile à mettre en place et une version multi-plateforme pour les utilisateurs plus techniques. Une fois installées, les deux versions fonctionnent de manière identique.

La version Windows facile à mettre en place :

Téléchargez le logiciel d'installation Windows depuis la Page de Téléchargement de POPFile,exécutez setup.exe et suivez les instructions mentionnées. Vous pouvez lancer POPFile à partir du Menu Démarrer (Démarrer > Programmes > POPFile > Run POPFile in background), ou bien redémarrer et il se chargera automatiquement. Nota Bene: rien n'apparaîtra sur votre écran ! Pour vérifier qu'il fonctionne correctement, lisez la section ci-dessous "Comment accéder à POPFile".

Exigences du système

Vous aurez besoin de plusieurs éléments pour faire marcher POPFile.

  1. La version la plus récente de POPFile
  2. Un compte courriel qui fonctionne avec le protocole POP3 (c'est le cas de la plupart des comptes, cependant vous devez ajouter des logiciels supplémentaires pour faire marcher POPFile avec des services basés sur le web tels que Hotmail et Yahoo!mail)
  3. Environ 2 méga-octets d'espace disque

Vous pouvez obtenir la version la plus récente de POPFile sous forme de fichier zip en haut de la liste affichée sur la page de téléchargement POPFile. Les utilisateurs de Windows voudront le fichier sous la section étiquetée Windows uniquement.

POPFile occupe un peu plus d'un méga-octet. Les listes de mots (appelées le "corpus") utilisées pour la classification de votre courriel prendront un supplément de place, qui peut varier suivant le nombre de courriels que vous utilisez pour entraîner POPFile et selon le nombre de catégories que vous créez.

Version Multi-plateforme

Chargez Perl en mémoire puis téléchargez le zip Perl de POPFile depuis la page d'accueil de POPFile, et décompressez-le vers le répertoire de votre choix. Sur la plateforme Windows, lancez-le avec la commande perl popfile.pl

Vérifier que POPFile fonctionne correctement

POPFile n'a pas une interface classique comme la plupart des programmes - vous y accédez grâce à votre navigateur. Ne vous inquiétez pas - personne ne peut y accéder, à moins que vous n'en décidiez autrement.

Pour charger à POPFile, allez à http://127.0.0.1:8080 , ou (pour la version Windows), cliquez sur le lien "POPFile User Interface" dans le Menu Démarrer. Nota Bene: Si durant l'installation de Windows, vous choisissez un point d'accès différent pour dialoguer avec POPFile, remplacez la partie "8080" avec le port que vous avez choisi.

Si vous recevez un message d'erreur, vérifiez que POPFile s'exécute vraiment et que vous avez le numéro correct du point d'accès (cela devrait être 8080, mais si vous l'avez changé, vous aurez besoin de changer l'adresse web à laquelle vous allez).

Si vous obtenez l'écran POPFile, félicitations ! A présent, il faut personnaliser la manière dont vous voulez trier votre courriel. Passons à la Configuration des Catégories. >>
