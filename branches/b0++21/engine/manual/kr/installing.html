팝파일 설치하기

팝파일은 2가지 버전으로 배포됩니다 - 설치가 쉬운 윈도우즈용과 연구용 크로스 플랫폼용. 일단 설치 되면 두 버전은 같습니다.

설치가 쉬운 윈도우즈 버전

팝 파일 다운로드 페이지에서 윈도우즈 인스톨러를 다운로드 하십시오. 그리고 setup.exe을 실행시키고 화면에 나오는 설명대로 따라하십시오. 시작 메뉴로부터 팝파일을 실행 시킬 수 있으며 (시작 > 프로그램 > POPFile > Run POPFile in Background), 또는 리부팅하면 자동으로 실행됩니다.
중요 사항: 화면에는 아무것도 표시되지 않습니다! 제대로 실행중인 것을 확인하는 방법은 아래의 "팝파일 정상실행 확인하기" 섹션을 읽으십시오.

시스템 요구사항

팝파일을 실행하기 위해서는 몇가지 필요한 사항이 있습니다.

  1. 최신 버전의 팝파일
  2. POP3를 지원하는 메일 계정(일반 사내/교내 계정은 대부분 지원합니다만 라이코스, 다음 메일 등의 웹 메일은 대부분 지원하지 않습니다.) 아웃룩이나 아웃룩 익스프레스를 사용하고 계신다면 POP3를 지원하는 계정이라 봐도 거의 틀림없습니다.
  3. 약 2MB의 디스크 공간

최신 팝파일 배포본은 팝파일 다운로드 페이지의 목록 맨 위에서 zip 파일 형태로 다운로드 하실 수 있습니다. 윈도우즈 사용자는 Windows Only 라는 부분에서 받으시면 됩니다.

팝파일 자체는 몇메가밖에 차지하지 않습니다. 이메일을 분류하는데 쓰이는 단어 목록(코퍼스라 불립니다)은 추가적으로 디스크 용량을 차지하는데, 이 용량은 팝파일이 처리하는 메일 양과 사용하실 버킷 수와 연관됩니다.

크로스 플랫폼 버전

참고: 프로그래머나 전산에 관심이 많으신 분들만 필요한 버전입니다. 일반 사용자는 전혀 필요 없습니다.

Perl을 설치후 실행시키시고, Popfile Perl zip 파일을 팝파일 홈페이지에서 받아 원하시는 디렉토리에 압축해제 하십시오. 윈도우즈에서는 perl popfile.pl이라는 명령어로 실행 시키십시오.

팝파일 정상실행 확인하기

팝파일은 다른 프로그램들처럼 일반적인 UI 화면이 없고 익스플로러 등의 웹 브라우저를 통해 조작하게 되어 있습니다. 걱정마십시오. 특별히 지정하지 않는 한 외부사용자들은 접근이 불가능합니다.

팝파일 화면을 보기 위해서는 http://127.0.0.1:8080를 익스플로러 등의 브라우저에 입력하거나, (윈도우즈 버전에서는) 시작 메뉴의 'POPFile User Interface'를 클릭하십시오. 중요 사항: 윈도우즈용 설치 과정 중 팝파일 제어를 다른 포트를 통해 하도록 설정하셨으면 위 링크중 "8080" 부분을 설정하셨던 값으로 대체하십시오.

웹 브라우저에서 에러메시지가 나오면 팝파일이 실제 실행중인지 확인하시고 포트번호가 정확한지 확인하십시오. (8080이 초기값입니다만, 만약 변경하셨으면 웹브라우저로 열어볼 주소도 바꾸셔야 합니다.)
